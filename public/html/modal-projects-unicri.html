Websites for UNICRI
(United Nations)

As Front-end Developer at UNICRI (United Nations agency), I was in charge of creating 3 promotional websites and 2 e-learning portals. The websites were for projects in the field of security and counter-terrorism, jointly created by United Nations and European Union. For all of them I made the designs and implemented the HTML/CSS pages. For some of them I also set up and customised the CMS. Some of those websites contained restricted data.

Technologies: HTML, CSS, JavaScript, jQuery, CMS, UX/UI Design, Photoshop.

  • Informational website (UNICRI counter-terrorism department)
  • Informational website (UNICRI Counter-Terrorism department)
  • E-learning website (UN/EU project on sharing counter-terrorism best practices)
  • E-learning website (UN/EU project on sharing major events security best practices)
  • Informational website (UN programme on security of major events)
  • Informational website (UN programme on security of major events)
  • Informational website (UN project on anti-corruption in the Andean countries)
